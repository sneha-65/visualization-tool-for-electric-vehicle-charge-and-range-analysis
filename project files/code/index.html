Electric Vehicle Charge & Range Analysis

🚗 Range Analysis

This section focuses on analyzing the driving range of electric vehicles. The range depends on factors such as battery capacity, vehicle efficiency, driving conditions, and terrain. The Tableau dashboard visualizes how different EV models perform in terms of distance covered per full charge.

🔌 Charging Analysis

Charging analysis compares various charging methods including fast charging, normal charging, and home charging. This section helps users understand charging time requirements and charging efficiency for different electric vehicles.

💰 Cost & Efficiency

This section evaluates the cost efficiency of electric vehicles by comparing cost per kilometer with conventional fuel vehicles. Factors such as electricity consumption, charging cost, and maintenance are considered to highlight the economic benefits of EVs.

🚙 EV Models

A comparative analysis of popular electric vehicle models is presented based on battery capacity, driving range, charging time, and efficiency. This helps users choose suitable EV models based on their requirements.

ℹ️ About the Project

The Electric Vehicle Charge & Range Analysis project aims to analyze and visualize important parameters of electric vehicles such as driving range, charging performance, and cost efficiency.

The data is stored in a MySQL database and visualized using Tableau Public. These interactive visualizations are embedded into a web application developed using the SB Admin 2 dashboard template.

This project demonstrates how data visualization and web technologies can be combined to provide meaningful insights for decision-making in the electric vehicle domain.
